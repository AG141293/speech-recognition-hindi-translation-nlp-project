# speech-recognition-hindi-translation-nlp-project
Python project for speech-to-text transcription, Hindi translation, and text file export using NLP libraries.
Project Overview

This project converts spoken audio into text using Speech Recognition, translates the recognized English text into Hindi, and saves the translated output into a text file.

It demonstrates a simple Speech-to-Text + NLP Translation Pipeline using Python.

Features

✔ Audio to text transcription
✔ English speech recognition using Google Speech API
✔ English to Hindi translation
✔ Save translated output to text file
✔ Error handling for recognition failures

Technologies Used
Python
SpeechRecognition
Google Speech API
Deep Translator
Jupyter Notebook
